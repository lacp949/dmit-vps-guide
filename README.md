## 三档线路，先搞清楚再买

进 DMIT 官网，第一眼会看到一堆缩写：Pro、EB、T1、LAX、HKG、TYO……初次看挺懵的。其实逻辑很简单，按线路质量分三档：

经济档。国际线路，没有针对中国大陆的专项优化。适合主要面向海外用户的业务，或者预算有限、先试试水的场景。流量给得比较足，超量后还有 1Gbps 限速保底，不会直接断流。最低年付 $36.9。

## 三个机房，各有什么侧重

### 洛杉矶（LAX）

套餐最全，价格在三地里最有竞争力，是大多数人的首选。

洛杉矶到国内的物理距离比香港远，延迟自然高一些——Pro 系列全国平均在 140-180ms 之间。但换来的是更便宜的价格：Pro TINY 入门款 $9.99/月，Eyeball TINY 同样 $9.99/月。

这里还有两个特殊产品线值得一提：

- **Premium Secure（sPro）**：在 CN2 GIA 基础上加了 Cloudflare Magic Transit DDoS 防护，适合游戏服务器、API 网关这类容易被打的场景。
- **Premium Unmetered（Pro.u）**：不限流量套餐，带宽有限制但流量不封顶，做视频站、CDN 源站很合适。

洛杉矶还有一款限量神车叫 Malibu（LAX.AN4.Pro.MALIBU），年付 $49.9，三网回程 CN2 GIA，每次上架很快就售罄。有货就是捡到了。

### 香港（HKG）

物理距离最近。国内用户访问香港节点，延迟通常在 10-30ms，比洛杉矶少一个数量级。

Premium 系列电信 CN2 GIA、联通 AS9929、移动 CMI，晚高峰测试数据持续稳定。对延迟有强需求的业务——实时通信、游戏、需要极低响应时间的 API——香港是最合适的选择。

代价是价格。香港 Pro TINY 入门款 $39.90/月，是洛杉矶同档的四倍。

香港 Eyeball（HKG.EB）是个不错的折中：CMI 线路，延迟也比洛杉矶好很多，价格比 Pro 系列低一截，从 $29.90/月起。

### 东京（TYO）

适合需要日本 IP，或者面向日本/东南亚用户的场景。延迟比香港稍高，但线路质量同样优秀——Premium 系列也是三网优化，CN2 GIA 回程。

有一点要注意：东京 Eyeball 系列已经下架，现在只剩 Premium 和 Tier 1 两条线，不要再去找 TYO.EB 了。

## 测试 IP

买之前先 ping 一下，看看从你所在地到对应机房的延迟是否符合预期：

- 洛杉矶：`154.17.2.2`
- 香港：`103.117.100.2`
- 东京：`169.197.142.2`

Windows 下命令行跑 `ping 154.17.2.2 -t`，看平均延迟和丢包率。晚上 9 点到 11 点之间测一次，白天再测一次，对比两次结果。如果晚高峰掉速超过一半，说明线路有拥塞。
